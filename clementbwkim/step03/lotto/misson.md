# 로또 만들기

로또를 구매 -> 발행 -> 표시하는 사이트를 만들어보겠습니다.

# 조건

각 로또 번호는 6개입니다.
돈을 넣으면 살 수 있는 만큼 로또를 구매할 수 있습니다. 구매 함수는 buyLottos() 입니다.
로또 1장의 가격은 1000원입니다.

## 예시

> buyLottos(14000); (버튼을 클릭하면 해당 함수가 실행됩니다.)
